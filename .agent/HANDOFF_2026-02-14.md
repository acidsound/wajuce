# Handoff Notes

This file used to describe the pre-migration native runtime. The current
direction is now:

- Native runtime: iPlug2-backed C++ WebAudio renderer.
- Native implementation files:
  - `native/engine/Source/WAIPlugEngine.h`
  - `native/engine/Source/WAIPlugEngine.cpp`
  - `native/engine/Source/ParamAutomation.h`
  - `native/engine/Source/RingBuffer.h`
- Dart FFI backend:
  - `lib/src/backend/backend_native.dart`
- C ABI:
  - `src/wajuce.h`

Use WebAudio spec behavior as the source of truth. Avoid restoring assumptions
from the old native graph when implementing missing nodes.
